# Lexical Syntax

## Comments

Line comments are preceded by `//`:

```fathom
// A line comment
```

## Doc comments

Module-level doc comments are preceded by `//!`:

```fathom
//! A module-level doc comment
```

Definition-level doc comments are preceded by `///`:

```fathom
/// A doc comment for a definitions
```
